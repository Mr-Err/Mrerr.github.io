二、github 配置SSH登录

  • 1、终端生成新的SSH密钥对
  • 2、打开公钥文件（通常是id_rsa.pub），复制其中的内容。
  • 3、登录到你的GitHub账户。
    • 点击右上角的头像，选择“Settings”（设置）。
    • 在左侧菜单中，选择“SSH and GPG keys”（SSH和GPG密钥）。
    • 点击“New SSH key”（新建SSH密钥）。
    • 在“Title”（标题）字段中输入一个描述性标题。
    • 在“Key”（密钥）字段中粘贴你复制的公钥内容。
    • 点击“Add SSH key”（添加SSH密钥）。
  • 4、验证
    • 终端执行
    • 正确信息
