二）、本地文件资源启动

  • 实现markdown图片链接显示
  • 安装插件：npm install hexo-renderer-marked
  • source下创建文件时，创建同名md文件的文件夹，将图片存放于文件夹内(注意：写图片路径时只需写图片名称即可，不要使用相对或绝对路径)
